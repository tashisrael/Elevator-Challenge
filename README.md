## 💻 Getting Started <a name="getting-started"></a>

You can get this project cloned and checkout how it works. Here is how you do this.

### Prerequisites

In order to run this project you need:

```sh
- A code editor like Netbeans or Eclipse
- If you use a code editor, you need JDK installed
- You can as well run it online using Repl.it (https://repl.it/languages/java)
```
 

### Setup

Clone this repository to your desired folder.


```sh
  git clone https://github.com/tashisrael/Elevator-Challenge.git
  cd Elevator-Challenge
```

Then open you code editor and open the cloned folder.
If you are to run it online, create a new folder in the online code editor and copy each class name and code one by one.



### Usage

You can build and run the project once you have done all the above.
